METAR Decoder

A METAR (METeorological Aerodrome Report) is a standardized aviation weather report used by pilots to observe weather conditions at an airport. Use this tool to translate a coded METAR into plain English.

Paste a complete METAR into the text box, then select Decode METAR.

  • Look up an airport’s International Civil Aviation Organization (ICAO) code here: airnav.com/airports.
  • Look up an airport’s METAR here: aviationweather.gov.
